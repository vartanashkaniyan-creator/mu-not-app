# اپ کدنویس موبایل

این اپ به شما امکان کدنویسی در HTML، CSS، و JavaScript را به راحتی می‌دهد.

## ویژگی‌ها:
- ویرایش HTML، CSS و JavaScript
- امکان اجرای کد و مشاهده نتایج
- رابط ساده و کاربردی

## نحوه استفاده:
1. اپ را دانلود و نصب کنید.
2. کد HTML، CSS و JavaScript را وارد کنید.
3. کد را اجرا کنید و نتیجه را مشاهده کنید.

توسعه‌دهنده: وارتان اشکانیان
