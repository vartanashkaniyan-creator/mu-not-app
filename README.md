# اپ ساز حرفه‌ای – App Builder

این پروژه یک اپلیکیشن ساز پیشرفته برای موبایل و وب است که از 22 فایل ماژولار تشکیل شده است.  
هدف آن ایجاد محیط SPA، مدیریت یادداشت، ماشین حساب و لیست کارها با رابط کاربری ساده و حرفه‌ای است.

## ویژگی‌ها

- پشتیبانی از چند زبان با ماژول i18n
- ذخیره‌سازی سریع و امن با Storage.js
- ناوبری SPA با Router.js
- قالب‌های آماده با Templates.js
- ماشین حساب، یادداشت و لیست کارها
- قابلیت پیش‌نمایش یادداشت‌ها
- آماده PWA و آفلاین با Service Worker
